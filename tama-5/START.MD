### How to start this sample:

### node version = 24.14.1
### npm version = 11.11.0
### yarn version = 4.13.0
### bun version = 1.3.12
### Successfully started on unix (MacOs 26.4.1)

1. **Installation:** `yarn install`
2. **Web:** `yarn web` (starts Next.js)
3. **Mobile:** `yarn native` (starts Expo)

**Important with errors:** 
if Metro (native/mobile) has bugs, use to start command: `npx expo start --clear`.
